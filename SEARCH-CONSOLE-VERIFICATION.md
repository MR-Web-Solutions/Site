# Search Console verification

1. Add `https://mrweb.co.za/` as a URL-prefix property in Google Search Console.
2. Choose HTML tag or HTML file verification.
3. Paste the supplied meta tag into every page head, or upload Google's supplied verification file to the site root.
4. In Bing Webmaster Tools, repeat the same process with Bing's supplied verification tag or file.

Verification codes are unique to the owner account and must not be guessed or committed before Google or Bing provides them.
